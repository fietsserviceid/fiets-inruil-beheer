
# Fiets Inruil — Beheer (cross-repo)

Deze repo beheert dealers en codes. De workflow schrijft naar **fietsserviceid/fiets-inruil-calculator/codes.json**.

## Vereist
- Repo secret in deze repo: `TARGET_REPO_PAT` (fine-grained PAT met **Contents: Read & Write** op fietsserviceid/fiets-inruil-calculator).

## Gebruik
- Open Issues in deze repo en kies één van de formulieren (label `beheer`).
- Of open `web/beheer/index.html` en klik op de knoppen.
